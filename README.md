# worklens-docs-public

Public documentation site for WorkLens (MVP-1).

## What this repo is
- Public docs only (safe for everyone to read)
- Built with Docusaurus and deployed via GitHub Pages
- Designed for agent-driven doc updates with guardrails

## Hard rules (important)
- ❌ No secrets (API keys, tokens, passwords)
- ❌ No customer data, screenshots with personal info, internal endpoints
- ❌ No “guaranteed compliance”, “100% secure”, “unhackable” claims
- ✅ Keep explanations high-level and user-focused

See: `DOCS_POLICY.md` and `AI_POLICY.md`.
